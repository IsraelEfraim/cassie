Conheça o CASSIE

O CASSIE é uma ferramenta web de código aberto que permite o mapeamento automático e análise da linha de costa em qualquer zona costeira do planeta. Para realizar esses procedimentos o CASSIE utiliza as ferramentas e imagens de satélite multi-espectrais disponibilizadas pela plataforma Google Earth Engine. O CASSIE tem acesso a coleção de imagens pertencentes às missões Landsat 5 (1984-2011), Landsat 7 (desde 1999 até o presente), Landsat 8 (desde 2013 até o presente) e sentinel-2 (desde 2013 até o presente). Como a plataforma Google Earth Engine é baseada em nuvem, a utilização do CASSIE requer acesso a internet, contudo o processamento de dados do CASSIE é realizado pela plataforma Google Cloud, não sendo por isso necessário um computador muito robusto para sua utilização.

Instruções para utilização do CASSIE

Projeto BAYSQUEEZE

O nome "BAYSQUEEZE" foi o acrônimo escolhido para o projeto científico intitulado: "Subida do nível do mar e a Baía da Babitonga: uma abordagem eco-morfodinâmica para prever e mitigar impactos". Este projeto foi financiado pela chamada pública do CNPq MCTIC/CNPq - N° 21/2017 - Pesquisa e Desenvolvimento em Ações Integradas e Sustentáveis nas Baías do Brasil, Linha de Pesquisa 2 - Outras Baías do Brasil. O objetivo do projeto BAYSQUEEZE é, através de uma abordagem multi-disciplinar e inovadora, analisar os impactos que a subida do nível médio do mar vai trazer para a Baía da Babitonga e explorar medidas de mitigação do tipo “construindo com a natureza”. O CASSIE foi desenvolvido a partir do projeto BAYSQUEEZE, realizando a meta de desenvolvimento de uma ferramenta para análise e gestão de dados geospaciais em ambientes costeiros como, praias, baías ou estuários.
